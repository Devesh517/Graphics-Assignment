# Graphics Algorithms using C++ (Turbo C++)

This project is a **menu-driven Computer Graphics application** developed in **C++ using Turbo C++ (`graphics.h`)**.  
It demonstrates the implementation of **fundamental graphics algorithms** such as line drawing, circle drawing, and shape construction using mathematical calculations.
